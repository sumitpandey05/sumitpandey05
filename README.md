# 💫 About Me:
I’m a Computer Science undergraduate at VIT Vellore with a strong interest in building practical, performance-focused software across web and systems domains. My work includes React-based frontend projects as well as low-level systems and embedded development using C, Python, Linux, and Raspberry Pi. I enjoy exploring areas like systems programming, embedded Linux, Edge AI, and full-stack development through hands-on projects that combine implementation, optimization, and real-world problem solving. I will continue building and learning about things that boost my curiosity.
